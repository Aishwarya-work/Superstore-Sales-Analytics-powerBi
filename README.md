# Superstore Sales Analytics - Power BI Project

## Project Overview
This project focuses on analyzing sales performance using Power BI. It provides insights into sales trends, profit performance, customer behavior, product performance, and regional contributions.

##  Objectives
- Analyze overall sales and profit
- Identify top-performing products and regions
- Perform Year-over-Year (YoY) analysis
- Build interactive dashboards
- Support data-driven business decisions

##  Tools Used
- Power BI
- Power Query
- DAX
- CSV Dataset

## Data Model
Star Schema Architecture:
- Fact_Sales
- Dim_Customer
- Dim_Product
- Dim_Region
- Dim_Date

## Data Preprocessing Steps
### 1. Data Loading

Imported CSV dataset into Power BI

### 2. Data Cleaning

Removed duplicates

Handled null values

Standardized text columns

Corrected data types

### 3. Transformation

Created calculated columns

Split and renamed columns

Built dimension tables

### 4. Date Table Creation

A custom date table was created to enable time intelligence analysis.

## Dashboard Features
- KPI Cards
- YoY Growth Analysis
- Quarter-wise Sales
- Category-wise Sales
- Region-wise Sales
- Product Performance
- Segment Analysis
- Interactive Filters

## Key Insights
- West region is the highest revenue contributor
- Office Supplies is the top-selling category
- Q4 has the highest sales growth
- Consumer segment generates maximum profit
